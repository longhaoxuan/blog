1.尽量减少改动时要编辑的地方

2.某些值互相依赖,要表现出来

3.确定不变的特效写死了

            .first{
                padding:6px 16px; //内边距离,撑起父类
                border: 1px solid #446d88; //边框(宽度,样式,颜色)
                background: #58a linear-gradient(#777777,#58a);
                /*背景颜色 linear-gradient渐变可以设置渐变方向 默认上下*/
                border-radius: 4px;//边框圆角
                box-shadow:00 -1px 1px grey;
                color: white;//字体颜色
                text-shadow: 0 -1px 1px #334466;
                /*text-shadow: h-shadow v-shadow blur color; (x偏移,y偏移,模糊度,颜色)*/
                font-size: 20px;//字体大小
                line-height: 30px;//行高
            }

优化

1.字体使用百分比,或em单位(如果希望根据父级元素自适,用em,如果根据根节点适应,用rem)

2.行高和字号的关系字号

3.确定变化的元素

            .second{
                padding:.3em .8em;
                border: 1px solid #446d88; 
                background: #58a linear-gradient(#777777,#58a);
                border-radius: .2em ;//边框圆角
                box-shadow:0 .05em 0.05em grey;
                color: white;//字体颜色
                text-shadow: 0 .05em .05em #334466;
                /*text-shadow: h-shadow v-shadow blur color; (x偏移,y偏移,模糊度,颜色)*/
                font-size: 150%;//字体大小
                line-height: 1.5;//行高
            }

4.颜色的变化,覆盖一层透明颜色改变明暗,然后改变被金个颜色即可

            background: green linear-gradient(hsla(0,0%,100%,.2),transparent)
            box-shadow:0 .05en .05em rgba(0,0,0,0.5)
